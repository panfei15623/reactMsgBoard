 留言板      

 出发点：    
 在学习React时候学习文档中有编写评论的教程，联想到可以增加分页、删除等功能，于是想到了QQ留言板有这些样式，便参照QQ留言板做了该项目。有了这些功能，React和以前学习的知识也能得到很好的应用，比如：分页（前端分页或者后端分页）、Nodejs中Express框架与Mysql的结合，上传文件的应用，Sass或者Less的应用等，最后在研究Webpack的时候，把webpack也加进来了。
 
 使用说明：    
 1、下载该文件，电脑上要有node环境，Express、React、Mysql、Webapck等模块     
 2、启动reactMsgBoard -> src -> server ->server.js文件   
 3、在浏览器中访问reactMsgBoard -> dist ->index.html即可  
 
 注：若是有自己的数据库，可以修改reactMsgBoard -> src -> server ->server.js文件中的mysql.createConnection里的相关配置，把数据存到自己的数据库中。  
  
 应用技术：
 其中应用到React框架实现模块化、ES6语法加载文件、Less编写css文件、Webpack打包压缩、Nodejs实现后台、Mysql数据库存储数据        
 
 功能：  
 1、实现留言功能，展示留言人姓名、留言时间、楼层、内容    
 2、实现删除留言功能，但是楼层不会变    
 3、实现分页   
 
 未完功能：  
 1、留言人头像可以自己上传   
 2、主人寄语可以编辑
